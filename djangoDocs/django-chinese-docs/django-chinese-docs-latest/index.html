这里有你需要知道的有关 Django 的一切。

Getting help¶

有问题吗？ 我们希望以下对大家有所帮助！

  • 试试 常见问题 – 它对许多常见的问题作了解答。
  • 寻找特定的信息？ 试试 文档索引, 模块索引 或者 文档目录 。
  • 搜索 django-users邮件列表,或者 提交一个问题 。
  • 在 #django IRC channel 上提问, 或者搜索 IRC logs 看下是否已经有人问过类似的问题。
  • 在我们的 ticket tracker 上报告 Django 的 bug 。
